hexo插入图片不能显示

1.安装插件(deyiBlog根目录打开Git Bash)

npm install hexo-asset-image --save

2.修改配置文件_config.yml的 post_asset_folder，false改为true

 

3.更改/node_modules/hexo-asset-image/index.js的代码

4.提交就好了

hexo d部署后总需要重新改域名问题

1.在source目录下（不是hexo根目录下），创建一个CNAME文件，可以用sublime创建，然后保存成（All files格式）

 

2.CNAME文件里写自己的域名

3.hexo g 重新生成一下

 

4.hexo d 部署到github上

 

5.完成
